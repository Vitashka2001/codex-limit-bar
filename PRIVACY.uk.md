# Приватність

[English](PRIVACY.md) · [Українська](PRIVACY.uk.md) · [Русский](PRIVACY.ru.md)

Codex Limit Bar працює локально на Mac користувача.

Застосунок:

- не збирає аналітику або телеметрію;
- не використовує власний сервер;
- не читає й не зберігає паролі, access token, refresh token або API-ключі;
- не запитує доступ до введення з клавіатури, запису екрана, мікрофона, камери або файлів користувача;
- отримує відомості про обліковий запис і ліміти лише через установлений локально `codex app-server`.

Під час зміни облікового запису Codex відкриває офіційний процес авторизації в браузері. Новий обліковий запис стає активним для інших локальних інструментів Codex на цьому Mac.

Користувач може зупинити опитування через пункт **Моніторинг лімітів** або закрити застосунок через пункт **Повністю вимкнути**.
