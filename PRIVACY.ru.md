# Приватность

[English](PRIVACY.md) · [Українська](PRIVACY.uk.md) · [Русский](PRIVACY.ru.md)

Codex Limit Bar работает локально на Mac пользователя.

Приложение:

- не собирает аналитику или телеметрию;
- не использует собственный сервер;
- не читает и не сохраняет пароли, access token, refresh token или API-ключи;
- не запрашивает доступ к вводу с клавиатуры, записи экрана, микрофону, камере или файлам пользователя;
- получает сведения об аккаунте и лимитах только через установленный локально `codex app-server`.

При смене аккаунта Codex открывает официальный процесс авторизации в браузере. Новый аккаунт становится активным для остальных локальных инструментов Codex на этом Mac.

Пользователь может остановить опрос через пункт **Мониторинг лимитов** или закрыть приложение через пункт **Полностью выключить**.
