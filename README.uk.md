Codex Limit Bar показує залишок доступного ліміту безпосередньо в рядку меню. Застосунок автоматично вибирає найкоротше доступне вікно, а в меню показує докладні 5-годинний і тижневий ліміти.

## Можливості

- відсоток і кольорова шкала в рядку меню;
- зелений колір при 50–100%, жовтий при 20–49% і червоний нижче 20%;
- докладні 5-годинний і тижневий ліміти з часом скидання;
- відображення активного облікового запису й тарифного плану;
- зміна облікового запису Codex через офіційний вхід у браузері;
- англійська, українська та російська мови інтерфейсу;
- ручне оновлення, призупинення моніторингу й запуск під час входу;
- нативна світла й темна тема.

## Вимоги

- macOS 13 Ventura або новіша;
- установлений [Codex](https://openai.com/codex/) або офіційне розширення Codex для VS Code/Cursor;
- виконаний вхід у Codex.

Codex Limit Bar використовує локальний `codex app-server`. Окремий API-ключ застосунку не потрібен.

## Установлення

1. Завантажте `Codex-Limit-Bar-1.1.1.dmg` зі сторінки [останнього релізу](https://github.com/Vitashka2001/codex-limit-bar/releases/latest).
2. Відкрийте образ і перетягніть **Codex Limit Bar** до `Applications`.
3. Запустіть застосунок. Його індикатор з’явиться праворуч у рядку меню.

Публічна збірка має локальний підпис, але не нотаризована Apple. Якщо macOS заблокує перший запуск, натисніть застосунок правою кнопкою, виберіть **Відкрити** й підтвердьте запуск. Це потрібно зробити лише один раз.

## Мова

Відкрийте меню, виберіть **Мова**, а потім **English**, **Українська** або **Русский**. Застосунок автоматично перезапуститься й запам’ятає вибір. До ручного вибору використовується бажана мова macOS.

## Керування

- **Моніторинг лімітів** тимчасово зупиняє фонове оновлення.
- **Змінити обліковий запис Codex...** відкриває офіційний вхід і змінює активний обліковий запис Codex на цьому Mac.
- **Запускати під час входу** вмикає або вимикає автозапуск.
- **Повністю вимкнути** завершує застосунок.

Щоб повністю вимкнути утиліту, спочатку зніміть позначку **Запускати під час входу**, а потім виберіть **Повністю вимкнути**. Для повторного запуску відкрийте застосунок з `Applications`.

## Приватність

Застосунок не читає й не зберігає паролі, токени або API-ключі. Він запускає встановлений локально Codex і отримує лише відомості про обліковий запис та ліміти. Докладніше: [PRIVACY.uk.md](PRIVACY.uk.md).

## Збірка з вихідного коду

Потрібні Xcode Command Line Tools і Swift 6:

```sh
swift test
./scripts/build-app.sh
```

Готовий застосунок з’явиться в `dist/Codex Limit Bar.app`.

Для створення Universal DMG і ZIP для Apple Silicon та Intel:

```sh
./scripts/package-release.sh
```
